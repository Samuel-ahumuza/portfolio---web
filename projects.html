Airport Baggage Tracker

HTML CSS JavaScript

Passengers at many airports, including Entebbe, often experience anxiety and inconvenience due to a lack of real-time information about their luggage. This can lead to long waits and confusion.

I'm developing a simplified web application that simulates a real-time baggage tracking system. The app will allow a user to enter a unique bag ID and receive updates on its status, from check-in to arrival at the carousel.

Cineplex Booking System

HTML CSS JavaScript

Cinemas today often involve long queues for tickets, limited information on available showtimes, and an outdated booking process. This creates inconvenience for moviegoers and inefficiencies for the cinemas.

I'm developing a user-friendly web application to streamline the movie-going experience. The app provides a real-time movie schedule, allows users to select a showtime, and enables seamless seat selection and booking.

E-commerce Supermarket Site

HTML CSS JavaScript DOM Manipulation

Modern retail relies on a seamless online shopping experience. This project simulates a simple e-commerce website for a supermarket, featuring product displays and ordering functionality.

The application features a dynamically loaded product catalog, filtering by category , and a fully functional shopping cart that tracks items, quantities, and calculates the running total.
